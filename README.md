Leaf_Cluster
===
a **experimental** game server base on leaf with cluster support, the code has not been completed yet.

Leaf
-----
A pragmatic game server framework in Go (golang).

Features
---------

* Extremely easy to use
* Reliable
* Multicore support
* Modularity
